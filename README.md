kvmator2
========
This web app been designed for DC Volia proposals.

It negotiates with Datacenter's equipment through SNMP and parse it into user-friendly view.

You can see it at [77.120.101.101](http://77.120.101.101)

Technolony stack:

1. Node.js
  * Express
  * MongoJS
  * Async
  * SNMP-native
2. MongoDB
3. Twitter Bootstrap
4. jQuery
